## Getting Started

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/EkaterinaTomarovskaya/room-booking.git
    ```

2.  **Install dependencies:**
    ```bash
    cd room-booking
    npm install
    ```

3.  **Run the app:**
    ```bash
    ng serve
    ```

4.  **Access the app:**
    * Frontend: http://localhost:4200/
